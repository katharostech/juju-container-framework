# Pager controls

|:-:            |:-:                    |
| **Action**    | **Key bindings**      |
|-              |-
| Scroll up     | `k`, Up Arrow         |
| Scroll down   | `j`, Down Arrow       |
| Page up       | Page Up, Backspace    |
| Page down     | Page Down, Spacebar   |
| Go to top     | `g`, Home             |
| Go to bottom  | `G`, End              |
| Help          | `h`, `?`
| Exit          | `q`, Esc, Enter       |
|-
