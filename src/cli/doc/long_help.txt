Show detailed documentation page ( similar to a man page ). Pager controls:

Scroll up:      k, Up Arrow
Scroll down:    j, Down Arrow
Page up:        Page Up, Backspace
Page down:      Page Down, Spacebar
Go to top:      g, Home
Go to bottom:   G, End
Exit:           q, Esc, Enter

You can also use less or cat instead of the default pager:

$ lucky --doc | less -R
$ lucky --doc | cat
